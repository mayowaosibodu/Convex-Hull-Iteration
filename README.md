# Convex-Hull-Iteration

Causal inference is known to be unreliable if it involves making predictions for extreme counterfactuals.

Here a technique is proposed: Convex hull iteration. By analyzing prediction accuracy of models on expanding convex hull subsets of the training data, it is possible to determine what model of a given set of models, is the most likely to predict correctly for extreme counterfactuals.

# Case in Point:
How different would the state of things in Saudi Arabia be if the country was a full democracy? Given historical data on countries of the world, answers based on this data is bound to be unreliable, primarily because there are no countries similar enough to Saudi Arabia, that are democracies.
With this technique however, it is possible to tell what model, of a given set of models, is closest to the truth. The really interesting questions are the weirdest ones, and this pushes us closer to being able to answer these questions.
